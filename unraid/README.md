# Tandarr Unraid Templates

Unraid Community Apps template for [Tandarr](https://github.com/liorbass/tandarr) — a LAN movie night picker.

## Installation

### Via Community Apps (Recommended)

Search for **Tandarr** in the Community Apps plugin.

### Manual Template Install

1. In Unraid, go to Docker tab
2. Click "Add Container"
3. Toggle to "Advanced View"
4. Set Template Repository URL to this repo
5. Fill in the required fields

## Files

- `tandarr/tandarr.xml` — Unraid CA template
- `tandarr/tandarr-icon.png` — App icon (256x256)
